Advance Selectors

Thus far, the basis selecters consist of:

  • tag = a, p, header, body, footer...
  • class = .container, .button, .box...
  • id = #main, #mail-list, #supernav...

Advance selectors consist of:

  • Selector stacking = a, strong { ... ; }
  • All Selector = * { ... ; }
  • Descendant Selector = li a { ... ; }, li li a { ... ; }
  • Direct Descendant Selector = .container > ul { ... ; }
  • Adjacent Selector = ul + p { ... ; }
  • Sibling Combinator = ul ~ p { ... ; }
  • Pseudo Classes = a:hover { ... ; }, textarea:focus { ... ; }
